# Parkinson's Voice Analysis Project(s): Manual and Modeled

This portfolio examines the voice quality of people with Parkinson's Disease and healthy controls. 

## Research Question(s):

1. How does the voice quality of patients with Parkinson's Disease (PwPD) differ from healthy individuals?
2. Can machine learning models be trained to accurately identify PwPD based on their voice?

You will find the following provided to fully understand the work that has gone into this:
---
## What you can find here

PGLscript: Praat script for acoustic feature extraction;
voice_measures: anonymized csv file generated from running Praat script;
PGLstats: R script to run MANOVA and t-tests with features;
PGLpaper: PDF of manuscript (ML interpretation pending); 
MLS_PDmodel: Python script for automatized feature extraction, correlations, and model comparisons.

## Data Availability

Raw audio files are not included due to ethical and privacy considerations. Audio files can be found in open dataset “Voice Samples for Patients with Parkinson’s
Disease and Healthy Controls” compiled by Prior et al. (2023).
